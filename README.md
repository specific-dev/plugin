# Specific Cursor Plugin

[Cursor](https://cursor.com) plugin for [Specific](https://specific.dev), a cloud platform designed for coding agents.

## Install

Install from the [Cursor Marketplace](https://cursor.com/marketplace) or run `/add-plugin` in Cursor.

You'll also need to install the Specific CLI. The Cursor agent can help you with this or you can run:

```sh
curl -fsSL https://specific.dev/install.sh | sh
```

## What's included

- **Skills** — teaches the Cursor agent about Specific and how to use it to build infrastructure
- **Rules** — auto-applies best practices when editing `specific.hcl` files
- **Commands** — includes commands for running local dev environments and deploying to production
