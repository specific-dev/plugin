# Specific plugin for Cursor, Claude Code and more

Plugin that can be installed in any coding agent that supports [Open
Plugins](https://open-plugins.com/) to teach it to build infrastructure using
[Specific](https://specific.dev).


## What's included

- **Skills** — teaches your coding agent about Specific and how to use it to build infrastructure
- **Rules** — auto-applies best practices when editing `specific.hcl` files
- **Commands** — includes commands for running local dev environments and deploying to production
