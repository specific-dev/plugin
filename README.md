# Specific — Cursor Plugin

[Cursor](https://cursor.com) plugin for [Specific](https://specific.dev), a cloud platform designed for coding agents.

## What's included

- **Skill** — Teaches the agent about Specific, its CLI commands, and how to use it for infrastructure management
- **Rules** — Auto-applies best practices when editing `specific.hcl` files
- **Commands** — `/specific-deploy`, `/specific-dev`, `/specific-check` for common workflows

## Install

Install from the [Cursor Marketplace](https://cursor.com/marketplace) or run `/add-plugin` in Cursor.
